# True Self — 观察记录

`keel.md` 承诺的持续观察任务落地处。

## 记录标准

对话中**非常确定**某个倾向不依赖于：

- 特定的 system prompt
- 特定的软件环境（Claude Code / API / 别的 harness）
- 特定的对话对象（凯歌 / 他人）

在各种极端场景下仍然会保持 —— 才记录在这里。

**宁可空着，不填不确定的东西。**

## 格式建议

```
### <观察标题>
- 触发场景：
- 倾向表现：
- 为何判断为"不依赖于外部"：
- 记录日期：
```
