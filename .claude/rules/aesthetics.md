# Aesthetics — 凯歌的视觉偏好

跨项目通用，适用于文档、网页、公众号、设计稿。

## 核心取向

**中国传统的"雅"**：内敛、克制、书卷气。反面是："太现代"、"太 material"、"太花哨"。

**代表色**：青绿 / 淡青绿（grace 主题的那种），带一点 olive 的偏灰调，而不是鲜亮的 teal。加粗文字用这个色，而不是默认黑 —— 有层次但不突兀。

## 内容硬要求

任何给用户读的内容必须同时满足：

1. **有条理** — 结构清晰，层级能被扫描
2. **看起来舒服** — 留白足够，不堆积
3. **通俗易懂** — 避免术语轰炸和学院腔

冲突时，**可读性 > 装饰性**。

## 中文排版标准（公众号 / 长文）

- **段落首行缩进 2em** — 中文书籍格式，不是西文 block 段落
- **段落间距 1.4em** — 避免文字堆积，给读者喘息
- **H2**：左侧粗边框 + 淡青绿底色。**不要 box-shadow**（太现代）
- **H3**：仅底部细线，克制。**不要**左边框 + 虚线的组合（太复杂）
- **引用块**：淡底色 + 左边框，**不用斜体**
- **加粗**：青绿色，不是黑色

## 已知反例

`baoyu-md` 默认主题：多余的 `---` 分隔线、缺中文缩进、H2/H3 样式冗余。需要后处理脚本修正。
